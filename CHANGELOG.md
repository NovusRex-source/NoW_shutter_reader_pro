# Changelog

All notable changes to Canon Shutter Counter Pro will be documented in this file.

## [2.0.0] - 2026-03-12

### Added
- ✨ Modern dark/light mode toggle with smooth transitions
- 🎨 Completely redesigned UI with responsive layout
- 📐 Rounded corners and modern card-based design
- 🖼️ Drag & drop support for CR3 files
- 📊 Real-time progress tracking for batch processing
- 🔍 Enhanced metadata display with color-coded output
- 💡 Professional tooltips and status indicators
- 📱 Responsive window design (900x600 minimum, 1280x800 default)
- 🎯 Tabbed interface for File/USB/Batch/Tools/Info
- 🚀 One-click build script for standalone EXE
- 📦 Distribution package creator

### Changed
- 🔄 Complete UI overhaul using ttkbootstrap
- 🎨 Migrated from basic tkinter to modern themed widgets
- 📝 Improved EXIF data formatting and presentation
- 🔧 Optimized ExifTool integration
- 📖 Enhanced documentation and help system

### Improved
- ⚡ Faster file processing
- 🎭 Better error handling and user feedback
- 🔐 More robust USB camera detection
- 📏 Cleaner code structure and organization

### Fixed
- 🐛 USB camera detection on Windows 11
- 🔧 ExifTool path resolution issues
- 📁 Batch processing memory optimization
- 🖼️ Icon loading on different systems

### Developer
- 👨‍💻 Noel Wangler - NoW Photography (noelwangler.ch)
- 📦 Ready for GitHub hosting
- 🤖 GitHub Actions workflow for automated builds
- 📚 Comprehensive documentation suite
